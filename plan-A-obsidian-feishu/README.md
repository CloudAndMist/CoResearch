# research-wiki

研究组本地/云端协同知识库 — 快速启动仓库模板

---

## 文件说明

| 文件/目录 | 说明 |
|-----------|------|
| `research-group-wiki-guide.md` | 📖 **完整部署指南**（分发给研究组成员） |
| `architecture.md` | 🏗️ 系统架构图与工作流程图（Mermaid 格式） |
| `CLAUDE.md` | ⚙️ Wiki 工作流定义文件（每位成员需个性化修改） |

## 快速开始

1. 阅读 `research-group-wiki-guide.md` 了解完整方案
2. 按照第 10 节「快速启动清单」逐步操作
3. 每位成员修改自己的 `CLAUDE.md` 中的个人信息

## 核心工具

- 本地：[Obsidian](https://obsidian.md/) + Git
- 云端：[飞书知识库](https://www.feishu.cn/) + 飞书机器人
- 自动化：GitHub Actions + Python + DeepSeek API
