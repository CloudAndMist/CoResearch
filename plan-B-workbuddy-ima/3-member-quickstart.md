# Plan B：成员快速上手指南

> **阅读时间**：5 分钟  
> **目标**：让你快速开始使用 IMA 管理研究资料

---

## 第一步：安装 IMA（2分钟）

### Windows 用户
1. 打开浏览器，访问 https://ima.qq.com
2. 点击「下载 Windows 版」
3. 下载完成后双击安装
4. 打开 IMA，用微信扫码登录

### Mac 用户
1. 打开浏览器，访问 https://ima.qq.com
2. 点击「下载 Mac 版」
3. 下载后拖动到「应用程序」文件夹
4. 打开 IMA，用微信扫码登录

---

## 第二步：加入共享知识库（1分钟）

你会收到管理员发来的**邀请链接**，点击即可加入。

或者：
1. 打开 IMA
2. 点击左侧「共享知识库」
3. 点击「加入知识库」
4. 输入管理员给的邀请码

---

## 第三步：开始使用（日常使用）

### 添加论文/资料

| 你想保存的内容 | 操作方法 |
|----------------|----------|
| **PDF 论文** | 直接把文件拖到 IMA 窗口里 |
| **网页链接** | 复制链接 → 在 IMA 中粘贴 → 回车 |
| **微信文章** | 点击文章右上角「···」→ 分享到 IMA |
| **随手笔记** | 点击「新建笔记」直接写 |
| **截图** | 截图后按 `Ctrl+V` 粘贴到 IMA |

### 使用 AI 帮你读论文

1. 在 IMA 中打开一篇论文
2. 点击底部的输入框
3. 输入你的问题，例如：
   - 「这篇论文的主要创新点是什么？」
   - 「请总结实验部分的关键结果」
   - 「这个方法和我之前读的 XX 论文有什么区别？」
4. IMA 会基于论文内容回答你

### 整理你的笔记

- 给笔记**打标签**：方便以后搜索
- **添加到共享库**：想让组里其他人看到的笔记，点击「添加到共享知识库」
- **写批注**：在 PDF 上直接标注重点

## 常见问题

**Q: IMA 收费吗？**  
A: 个人使用免费，有 30GB 存储空间，对一般研究来说够用。

**Q: 我的笔记别人能看到吗？**  
A: 个人知识库只有你自己能看到。只有明确「添加到共享知识库」的内容，组员才能看到。

**Q: 离线时能用吗？**  
A: 可以！个人知识库支持离线查看。共享知识库需要联网。

**Q: 怎么导出笔记？**  
A: 右键点击笔记 → 导出，支持 Markdown 或 PDF 格式。
