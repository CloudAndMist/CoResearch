## 快速开始

### 管理员（20分钟部署）

1. **配置 IMA 共享知识库**（10分钟）
   - 创建共享知识库
   - 邀请成员加入
   - 获取 IMA API Key

2. **配置 WorkBuddy 自动化**（10分钟）
   - 设置每周六 16:30 Lint 检查
   - 设置每周六 17:00 生成组会议题
   - （可选）接入微信通知

👉 详细步骤见：[2-setup-guide.md](./2-setup-guide.md)

### 成员（5分钟上手）

1. **安装 IMA**（2分钟）
   - 访问 https://ima.qq.com 下载安装
   - 微信扫码登录

2. **加入共享知识库**（1分钟）
   - 点击管理员发送的邀请链接

3. **开始使用**（随时）
   - 拖拽论文到 IMA
   - 使用 AI 问答功能
   - 周日18:30组会时一起查看自动生成的议题

👉 详细步骤见：[3-member-quickstart.md](./3-member-quickstart.md)

| 优势 | 说明 |

| ✅ **零配置** | 成员只需安装 IMA，无需配置 Git、Obsidian 等 |

## 工作流程

```
周一 ~ 周六：
├── 成员阅读论文 → 保存到 IMA 个人知识库
├── 做笔记、标注重点
└── 将需要分享的笔记「添加到共享知识库」

周六 16:30：
└── WorkBuddy 自动执行 Lint 检查
    ├── 检测重复内容
    ├── 检查信息完整性
    └── 生成质量报告（仅管理员可见）

周六 17:00：
└── WorkBuddy 自动生成组会议题
    ├── 读取本周共享库新增内容
    ├── 调用 LLM 分析生成议题
    └── 保存到 IMA 共享库「组会讨论」文件夹

周日18:30组会时：
├── 打开 IMA 共享知识库
├── 查看「本周组会议题」笔记
├── 基于议题进行讨论
└── 讨论后直接在 IMA 中记录会议纪要
```

## 常见问题

**Q: 为什么去掉飞书？**  
A: 飞书仅用于消息推送，而组会时本来就会一起查看议题。去掉飞书可以简化工具链，降低配置复杂度。

**Q: 成员如何知道议题已生成？**  
A: 1) 组会时一起查看（推荐）；2) 管理员可配置 WorkBuddy 微信通知。

**Q: 只用 IMA 可以吗？**  
A: 可以！IMA 共享知识库本身就是云端协作方案，配合 WorkBuddy 自动化即可实现完整功能。

**Q: IMA 免费版够用吗？**  
A: 个人知识库 30GB，对一般研究组足够。共享库无明确限制。

**Q: 成员需要付费吗？**  
A: 不需要。IMA 免费版 + WorkBuddy 免费使用。
